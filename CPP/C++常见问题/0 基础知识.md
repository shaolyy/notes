# 基础知识一

**static 关键字的作用**
- 全局静态变量
  - 关注静态存储区、初始化、和作用域。
  - 改变链接属性，由默认extern改为internal，不改变存储区和作用域
- 局部静态变量
  - 改存储类型由动态变量改为静态变量，程序执行前创建
- 静态函数
- 类静态成员
- 类中静态函数

**C++和C的区别**
- C++为面向对象，C面向过程的结构性语言
- 有封装、继承和多态
- C++增强类型安全
- C++支持泛型编程，如类模板、函数模板

**C++中四种cast转换** 

四种类型转换为static_cast, dynamic_cast, const_cast, reinterpret_cast

- const_cast:用于将const变量转为非const
- static_cast:各种隐式转换，比如非const转const，void*转指针等
- dynamic_cast:用于动态类型转换。只能用于含有虚函数的类，用于类层次间的向上和向下转化。只能转指针或引用。
- reinterpret_cast:几乎什么都可以转，比如将int转指针，可能会出问题，尽量少用
- C的强制转换无错误检测，更容易出错

**C/C++ 中指针和引用的区别**

- 指针有自己的一块空间，而引用只是一个别名
- .使用sizeof看一个指针的大小是4/8，而引用则是被引用对象的大小
- 初始化不同，指针可以改变指向对象
- 指针要解引用
- 返回动态内存分配对象，需要使用指针，引用可能引发内存泄漏
